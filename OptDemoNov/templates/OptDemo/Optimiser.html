Example of an Optimisation Process


The example proceeds by calling a Python script (on the background) that will run a Genetic Algorithm to solve a vehicle routing problem.

The vehicle routing problem in particular is one with capacity constraints on the trucks and the cities to visit are randomly generated. The distances are calculated using the Euclidean norm.

Please press the button, wait not so long and you will be presented with an image of the routing optimised by the algorithm. If you want to run it again, please come back to this page and press the button to see how a new problem instance is solved with the help of a Genetic Algorithm.
