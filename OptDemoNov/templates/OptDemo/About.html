About the Demo


This demo has been prepared to illustrate the possibilities of web-development in the context of Reverse Logistic applications.

It is not intended to have a final specification implemented but to serve as a vehicle to discuss industry requirements. In particular, as the model for reverse logistic is being developed as part of ongoing research, it has been replaced in this demo by a Genetic Algorithm optimiser example for a capacitated vehicle routing problem.

Some of the technologies used in this demo are:

  • Django
  • Python
  • JavaScript
  • JQuery
  • Three.js
  • D3.js
  • SqlLite3
  • Boostrap

Any questions relating to the demo and its implementation can be directed to Jose Saavedra-Rosas at charango.chile@gmail.com.
